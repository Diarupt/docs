### Install

The Javascript SDK is available as a package on [npm](https://www.npmjs.com/package/diarupt). You can install using npm, yarn or any other compatible package manager.

<CodeGroup>

```bash npm 
npm install diarupt
```

```bash yarn
yarn add diarupt
```
</CodeGroup>
